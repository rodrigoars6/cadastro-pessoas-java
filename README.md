# cadastro-pessoas-java
# 📋 Sistema CRUD de Pessoas em Java

Projeto desenvolvido em Java utilizando **ArrayList**, **CRUD**, **Leitura e Gravação de Arquivos TXT** e **Programação Orientada a Objetos**.

## 🚀 Funcionalidades

O sistema possui um menu interativo no terminal com as seguintes opções:

```text
1 - Incluir Pessoa
2 - Alterar Pessoa
3 - Excluir Pessoa
4 - Listar Pessoas
9 - Sair

📌 Funcionalidades implementadas

✅ Cadastro de pessoas
✅ Alteração de dados
✅ Exclusão de registros
✅ Listagem de pessoas
✅ Uso de ArrayList
✅ Persistência de dados em arquivo .txt
✅ Leitura automática do arquivo ao iniciar o programa
✅ Salvamento automático ao sair do sistema
✅ Campo adicional cargo na classe Pessoa

🛠️ Tecnologias utilizadas
* Java
* POO (Programação Orientada a Objetos)
* ArrayList
* Scanner
* BufferedReader
* BufferedWriter
* FileReader / FileWriter

📂 Estrutura do Projeto
ProjetoCRUD/
│
├── Main.java

└── pessoas.txt

🧠 Estrutura do arquivo TXT

Os dados são armazenados no arquivo pessoas.txt no seguinte formato:

Ana,35,Professora
Bruno,20,Programador
Carlos,40,Analista
